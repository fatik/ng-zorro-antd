Developer Integration (3 Steps)
================================

Step 1: Copy Package
--------------------
cp -r serviceant-package /their-angular-app/

Step 2: Update angular.json
----------------------------
Add stylePreprocessorOptions:
{
  "includePaths": ["node_modules", "src"]
}

Step 3: Import Theme
--------------------
Create src/styles.less:
@import 'styles/themes/serviceant.less';
@import 'ng-zorro-antd/ng-zorro-antd.less';

âœ… Done! Run npm start
